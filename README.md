Proyecto ECO-PET: Protocolo de Trazabilidad RutaPet (RPET)
Ubicación: Sincelejo, Sucre, Colombia.
Estado: Fase de Validación (Mainnet Activa).
🌍 Visión General
ECO-PET es una plataforma de economía circular que utiliza la tecnología blockchain para certificar la recuperación de residuos plásticos en el departamento de Sucre. Mediante el token RPET, transformamos la logística de recolección informal en un activo digital trazable y auditable.
Sistema de Verificación (Prueba de Impacto)
Para garantizar la transparencia ante organismos multilaterales y fundaciones, cada fracción de token en circulación está respaldada por:
Registro Físico: Pesaje verificado en estaciones de acopio asociadas.
Evidencia Digital: Registro fotográfico inmutable disponible en nuestra Página Oficial de Facebook.
Auditoría On-chain: Cada movimiento del token puede ser rastreado en el explorador de Solana.
